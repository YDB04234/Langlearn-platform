# 🚀 部署到线上 - 获得永久网址

## 📋 方法一：使用 Vercel（最简单，推荐）

### 步骤 1：访问 Vercel
打开浏览器访问：https://vercel.com

### 步骤 2：登录
使用你的 GitHub 账号登录

### 步骤 3：导入项目
1. 点击 **"Add New..."** → **"Project"**
2. 选择 **Langlearn-platform** 仓库
3. 点击 **"Import"**

### 步骤 4：部署
1. 保持默认设置
2. 点击 **"Deploy"**
3. 等待约 1 分钟

### 步骤 5：获得网址
部署完成后，你会得到一个永久网址，类似：
`https://langlearn-platform.vercel.app`

---

## 🌐 方法二：使用 Netlify（同样简单）

### 步骤 1：访问 Netlify
打开浏览器访问：https://app.netlify.com

### 步骤 2：登录
使用 GitHub 账号登录

### 步骤 3：从 Git 导入
1. 点击 **"Add new site"** → **"Import an existing project"**
2. 选择 GitHub
3. 选择 **Langlearn-platform** 仓库
4. 点击 **"Deploy site"**

### 步骤 4：获得网址
部署完成后，你会得到一个永久网址！

---

## 📦 方法三：使用 GitHub Pages

### 步骤 1：配置项目
1. 在项目根目录创建 `vite.config.ts`（已有）
2. 确认 `base` 设置正确

### 步骤 2：构建并上传
我们已经构建完成（`dist` 目录）

### 步骤 3：在 GitHub 上设置
1. 访问：https://github.com/YDB04234/Langlearn-platform/settings/pages
2. **Source** 选择：`Deploy from a branch`
3. **Branch** 选择：`main`
4. **Folder** 选择：`/dist`
5. 点击 **"Save"**

### 步骤 4：等待部署
约 1-2 分钟后，你的网址就准备好了：
`https://ydb04234.github.io/Langlearn-platform`
