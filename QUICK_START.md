# ===========================================
# 多语言学习平台 - 快速上手指南
# ===========================================

## 🚀 在本地运行项目

### 第一步：解压项目
```bash
tar -xzvf langlearn-platform.tar.gz
cd langlearn-platform
```

### 第二步：安装依赖
```bash
npm install
```

### 第三步：启动开发服务器
```bash
npm run dev
```

打开浏览器访问 http://localhost:5173 就能看到项目了！

---

## 📤 推送到 GitHub

### 第四步：初始化 Git 并提交代码
```bash
git init
git add .
git commit -m "feat: 多语言学习平台 - 完整功能实现"
```

### 第五步：推送到 GitHub
```bash
git remote add origin git@github.com:YDB04234/langlearn-platform.git
git branch -M main
git push -u origin main
```

### 第六步：验证
访问 https://github.com/YDB04234/langlearn-platform 查看你的代码！

---

## ⚠️ 重要提醒

在执行第五步之前，请先：
1. 打开浏览器访问 https://github.com/new
2. 创建名为 `langlearn-platform` 的仓库（不要勾选 README）
3. 然后再执行推送命令
